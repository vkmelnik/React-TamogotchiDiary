## Запуск

Проект запускается командами

    npm install
    expo start

При необходимости нужно установить следущие компоненты:

    react-native-chart-kit

    react-native-gesture-handler

    react-native-dialog

    react-native-stars

    moment

    react-native-calendar-picker
